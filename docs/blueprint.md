# **App Name**: Universal Image Toolkit

## Core Features:

- Image Compressor: Compresses image files to reduce their size without significant quality loss, optimizing them for web use.
- Image Resizer: Allows users to change the dimensions of images, making them suitable for different display sizes or platforms.
- Image Format Converter: Converts images between various formats (JPG, PNG, WebP) to ensure compatibility and efficient storage.
- Image Crop Tool: Enables users to crop images to focus on specific areas or adjust the composition.
- Image to Base64 Converter: Transforms images into Base64 strings, which can be embedded directly into HTML or CSS files.
- Color Picker from Image: Extracts color values from any point in an uploaded image, providing users with RGB or HEX codes.
- Metadata (EXIF) Viewer: Displays the EXIF data embedded in image files, providing information about the image's settings and camera details.
- Background Remover: Detects and removes the background from images, using a simple client-side masking technique.
- Image Rotator/Flipper: Rotate or flip images horizontally or vertically with instant results.

## Style Guidelines:

- Primary color: Soft blue (#72BCD4) inspired by a clean aesthetic.
- Background color: Off-white (#F9F9F9) to maintain a bright, user-friendly environment.
- Accent color: Light green (#90EE90) for positive actions and highlights.
- Body and headline font: 'PT Sans' for clear and accessible readability.
- Use clean, simple icons for each tool, aligning with a modern design aesthetic.
- Implement a card layout with ample spacing to avoid clutter and enhance usability.
- Employ subtle transitions and feedback animations to improve user interaction and visual appeal.
